# Contra やりとり記録

クライアントごとにファイルを作成し、やりとりの内容を記録します。

## アカウント情報

| 項目 | 内容 |
|------|------|
| プラットフォーム | [Contra](https://contra.com/) |
| 手数料 | 0%（フリーランサー側） |
| プラン | Free |
| サービスURL | https://contra.com/s/7qXPsYcR-website-loc... |

## ファイル命名規則

- `クライアント名.md` で保存
- 例: `john_smith.md`, `tanaka_taro.md`

## 記録テンプレート

各ファイルは `_template.md` をコピーして使用してください。
